"Solución web para el registro y control de préstamos de libros en una biblioteca"

El presente proyecto busca desarrollar una solución web para una biblioteca, esta solución permitirá a los usuarios de la biblioteca registrar una cuenta, visualizar los libros disponibles para préstamo, reservar libros y notificar los retrasos en la devolución de los libros.

Objetivo general

- Desarrollar una solución web para la reserva de libros para una biblioteca.

Objetivos Especificos

- Analizar el proceso que se lleva a cabo para el préstamo de libros en una biblioteca. - Construir una solución web que permita la reserva de libros, visualizar los libros disponibles y notificar en caso de retraso. - Probar el funcionamiento de la solución web y validar el cumplimiento de los requisitos.

Alcance

Etapa 1:

Analizar el proceso que se lleva a cabo para el préstamo de libros en una biblioteca.

- Identifica el proceso de préstamo de libros en la biblioteca de la Universidad Nacional de Loja por medio de una entrevista.

- Identificar los requisitos funcionales y no funcionales de la solución web aplicando el análisis de requerimientos.

- Modelar el proceso de préstamo de libros.

- Documentar los requisitos funcionales y no funcionales

Etapa 2

Construir una solución web que permita la reserva de libros, visualizar los libros disponibles y notificar en caso de retraso.

- Elaborar el diseño de la solución web usando el lenguaje de modelado UML.

- Diseñar el modelo arquitectónico de la solución web.

- Codificar la solución web en base a su diseño y modelo arquitectónico.

Etapa 3

Probar el funcionamiento de la solución web y validar el cumplimiento de los requisitos.

- Validar el correcto funcionamiento de la solución web.

- Validar el cumplimiento de los requisitos funcionales y no funcionales.

- Realizar pruebas para encontrar y solucionar posibles errores.
